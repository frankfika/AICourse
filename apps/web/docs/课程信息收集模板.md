# AI Academy 课程信息收集模板

本文档用于收集课程和学位信息，请按照以下格式填写。

---

## 一、单门课程 (Course)

每门课程需要提供以下信息：

### 必填项

| 字段 | 说明 | 示例 |
|------|------|------|
| **课程名称** | 简洁明了的课程标题 | 大模型开发入门 |
| **课程描述** | 1-3句话介绍课程内容和价值 | 从零开始学习大模型开发，掌握 Prompt Engineering 和 Fine-tuning 技术 |
| **讲师姓名** | 课程讲师名称 | 张教授 |
| **课程时长** | 总时长（分钟） | 120 |
| **难度等级** | Beginner / Intermediate / Advanced / Expert | Beginner |
| **课程类型** | free（免费）/ paid（付费）/ charity（公益） | free |
| **课程价格** | 付费课程填写价格（人民币），免费填 0 | 99 |
| **课程标签** | 2-5个关键词标签 | AI, 大模型, Prompt, 入门 |
| **学习要点** | 3-5个学完后能掌握的技能点 | 见下方示例 |
| **封面图片** | 课程封面图（建议尺寸 1280x720，支持 JPG/PNG） | 上传图片文件 |

### 可选项

| 字段 | 说明 | 示例 |
|------|------|------|
| **视频链接** | 课程视频URL（支持 YouTube、Bilibili 或本地视频） | https://www.bilibili.com/video/BV1xxx |
| **课程资源** | 附加资源列表（PDF、代码、链接等） | 见下方格式 |

### 学习要点示例

```
1. 理解大模型的基本原理和应用场景
2. 掌握 Prompt Engineering 的核心技巧
3. 学会使用主流大模型 API
4. 完成一个简单的 AI 应用开发
```

### 课程资源格式

每个资源包含：
- **资源名称**：资源标题
- **资源类型**：pdf / code / link
- **资源链接**：下载或访问链接

示例：
```
资源1：课程 PPT | pdf | https://xxx.com/slides.pdf
资源2：示例代码 | code | https://github.com/xxx/demo
资源3：参考文档 | link | https://docs.xxx.com
```

## 二、Nano Degree（学位/学习路径）

每个学位需要提供以下信息：

### 必填项

| 字段 | 说明 | 示例 |
|------|------|------|
| **学位名称** | 学位标题 | AI 全栈工程师 |
| **学位描述** | 2-3句话介绍学位内容和目标 | 系统化学习 AI 技术栈，从基础到实战，成为全栈 AI 工程师 |
| **包含课程** | 该学位包含的课程列表（按学习顺序排列） | 课程1、课程2、课程3... |
| **学位价格** | 人民币价格，免费填 0 | 599 |
| **学位类型** | free（免费）/ paid（付费）/ charity（公益） | paid |
| **学习要点** | 3-5个完成学位后能掌握的能力 | 见下方示例 |

### 可选项

| 字段 | 说明 | 示例 |
|------|------|------|
| **封面图片** | 学位封面图（建议尺寸 1280x720） | 上传图片文件 |
| **图标样式** | shield / sparkles / layers / terminal / cpu / code / globe | sparkles |

### 学习要点示例

```
1. 完整掌握 AI 应用开发全流程
2. 具备独立开发 AI 产品的能力
3. 理解主流大模型的原理与应用
4. 获得项目实战经验
```

## 四、注意事项

1. **图片要求**
   - 格式：JPG 或 PNG
   - 建议尺寸：1280 × 720 像素
   - 文件大小：建议不超过 2MB

2. **视频要求**
   - 支持平台：YouTube、Bilibili
   - 也支持上传本地视频文件（MP4、WebM）

3. **内容规范**
   - 课程描述和学习要点请尽量简洁明了
   - 标签使用常见技术关键词，方便搜索

4. **提交方式**
   - 文字信息可以通过文档/表格提交
   - 图片和视频文件请单独打包发送
