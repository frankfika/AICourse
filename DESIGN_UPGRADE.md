## 🎬 动画效果

### 全局动画

#### 1. **Fade In (淡入)**
```
- 从透明到不透明
- 同时向上移动 20px
- 持续时间: 0.8s
- 缓动函数: ease-in-out
```

#### 2. **Blob Animation (光斑动画)**
```
- 7秒循环
- 3个关键帧
- 位移 + 缩放
- 交错延迟 (0s, 2s, 4s)
```

#### 3. **Floating (浮动)**
```
- 3秒循环
- 上下浮动 20px
- 缓动函数: ease-in-out
- 无限循环
```

#### 4. **Gradient Animation (渐变动画)**
```
- 15秒循环
- 400% 背景尺寸
- 位置变化动画
- 无限循环
```
