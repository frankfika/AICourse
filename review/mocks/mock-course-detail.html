课程简介

Agent 是 LLM 时代最被高估、也最被低估的能力。我们见过太多 "Hello World Agent" 教程,但真正能在生产环境跑起来的,需要理解 tool calling 的不确定性、memory 的边界、chain 的可观测性。 本课程从 5 个真实项目出发:命令行助手、PDF 问答、浏览器自动化、代码审查 Bot、跨工具研究助手。

你将学到

  • ✓ Prompt Engineering 的结构化方法(ReAct / CoT / ToT)
  • ✓ Function Calling 的 JSON schema 设计与错误处理
  • ✓ Memory 选型:ConversationBuffer / Summary / VectorStore
  • ✓ 工具集成:Web 搜索 / 数据库 / API / 文件系统
  • ✓ 可观测性:LangSmith / 自建 trace 的取舍

先修知识

需要 Python 基础 + 至少用过 OpenAI / Anthropic API。无需 LangChain 经验。
